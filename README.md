# Chat App

A minimalistic mobile chat app built with React Native, Expo, and Google Firestore.

## Features
- Join chat with a custom name and background color
- Real-time messaging with offline support
- Send images (from library or camera)
- Share your location (map view)
- Anonymous authentication via Firebase
- Data stored in Firestore and locally
- Accessible and screen reader friendly

## Tech Stack
- React Native
- Expo
- Google Firestore & Firebase Auth
- Firebase Cloud Storage
- Gifted Chat library

## Usage
1. Clone the repo
2. Run `npm install`
3. Start with `npm start` (Expo)
